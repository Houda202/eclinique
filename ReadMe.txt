# E-Clinique

## Description
E-Clinique est une application web permettant de gérer les rendez-vous dans une clinique médicale. Elle comprend des fonctionnalités telles que la prise de rendez-vous, la gestion des utilisateurs et l'administration de la clinique.

## Fonctionnalités
- Prise de rendez-vous en ligne
- Système d'authentification pour les administrateurs, médecins, assistants, et patients
- Gestion des spécialités médicales
- Interface utilisateur réactive et conviviale

## Structure du Projet
- `assets/images` : Images utilisées dans le projet
- `index.html` : Page d'accueil
- `about.html` : Page à propos
- `services.html` : Page des services
- `contact.html` : Page de contact
- `appointment.html` : Page de prise de rendez-vous
- `admin.html` : Page d'administration
- `admin_dashboard.php` : Tableau de bord pour l'administration
- `assistant.html` : Page pour les assistants
- `billing.html` : Page de facturation
- `doctor.html` : Page pour les médecins
- `form.html` : Formulaire générique
- `login.html` : Page de connexion
- `login.php` : Script de traitement de la connexion
- `patient.html` : Page pour les patients
- `script.js` : Fichier JavaScript pour les interactions dynamiques
- `speciality.html` : Page des spécialités
- `style.css` : Fichier de styles CSS
- `submit.php` : Script de traitement de soumission générique
- `submit_appointment.php` : Script de traitement de la prise de rendez-vous
- `submit_demande_rendez_vous.php` : Script pour traiter les demandes de rendez-vous
